# PageTable

The PageTable component is used for adding a table to a page.
See the <a href="docs/guides/PageTableGuide.md">Table Guide</a>.

**Example**

```tsx
<Page>
  <PageLayout>
    <PageHeader ... />
    <PageTable ... />
  </PageLayout>
</Page>
```
